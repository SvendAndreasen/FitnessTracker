# Fitness Tracker

Local React app to register workouts. Data is stored in your browser (`localStorage`).

## MVP fields

- **Required:** exercise name, date
- **Optional:** sets, reps, weight (kg), duration (minutes), notes

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
